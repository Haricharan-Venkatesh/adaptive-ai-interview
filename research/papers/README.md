# Research Papers Index

Store copies or references to all papers this project builds upon.

## How to Add a Paper

1. Name the file: `YYYY_FirstAuthorLastName_ShortTitle.pdf`
2. Add an entry to this README
3. Add a citation to `docs/references.bib`
